# Vim Config

## Usage 

- install vim

- clone this repo

```bash
git clone https://github.com/hellopao/Vim
```

- copy all the files in Vim to Vim installed directory

- install vundle

```bash
cd $Vim
git clone https://github.com/gmarik/vundle  ./vimfiles/bundle/vundle
```

- install vim plugins

```bash
:PluginInstall
```
